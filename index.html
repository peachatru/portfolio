WatchTime

In a team of three, I contributed to creating the Progressive Web App that allows users to register or login to the app, browse various genres, save movies to their watchlist, and add reviews similar to Goodreads.



The technologies used for this app are Node.js/Express, Nginx Proxy, Docker, and MySQL. User objects such as their watchlist, reviews, etc. are stored in the MySQL database. We've also began implementing offline functionality through Service Workers caching all user activity done with internet connection.



Moving Forward

I plan on increasing security by implementing JWT Authentication and improving on offline functionality.

'Paw'modoro Timer

By randomly generating various hex codes and cat facts, the users are able to have a small dose of serotonin before they embark on their study session! Applying the Pomodoro study method, the application has a timer that's default value is 25 minutes, and an alarm will go off when time is up.



Compromised of HTML, CSS, and JavaScript, I extended this project to become a Google Chrome Extension where users can access it on the side!



Moving Forward

For my next steps, I would love to flesh out a To-do application for this project, so that users are able to store their daily tasks and keep track of what they've completed during their work session.

Pixel Pal

As a Progressive Web App, Pixel Pal allows users to create their artwork through pixels, adjust the grid size, erase, undo/redo, and save their image as either a jpeg, png, or svg file. It also has a responsive interface for both IOS/Android Devices!



Moving Forward

My next steps are to expand this project that features an AI Image Generator. The AI will create a pixelated image when prompted with text or an image by the user.

Howler

Inspired by the social media app "Twitter", the user is able to login through JWT Authentication, create 'howls', follow or unfollow other users, and to view their own profile. The howls created by the user will be shown on both the homepage as well as their profile page.



The technologies used for this app are Node.js/Express, Nginx Proxy, Docker, and MySQL.



Moving Forward

My next steps with this project is to implement a search feature that allows the user to search for other users and follow them. I'd also like to implement a chat feature where users can send 'howls' or chat with their friends as well.
